# Willians Tavares - Portfólio

Portfólio pessoal e blog de Willians Tavares. 

## 🚀 Tecnologias

- **Next.js 14** - Framework React para produção
- **TypeScript** - Tipagem estática
- **React 18** - Biblioteca de UI
- **React Markdown** - Renderização de conteúdo Markdown
- **Gray Matter** - Parser de front matter para posts
- **RSS** - Geração de feed RSS

## 📁 Estrutura

```
├── components/     # Componentes React (Layout, Header, Footer)
├── lib/            # Funções utilitárias (posts)
├── pages/          # Páginas do Next.js
├── posts/          # Artigos em Markdown
├── public/         # Assets estáticos (imagens, fontes)
├── scripts/        # Scripts (geração de RSS)
└── styles/         # Estilos CSS
```

## 🛠️ Como executar

```bash
# Instalar dependências
yarn install

# Executar em desenvolvimento
yarn dev

# Build para produção
yarn build

# Executar build de produção
yarn start
```

O site estará disponível em `http://localhost:3000`
